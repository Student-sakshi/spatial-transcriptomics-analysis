# Spatial Transcriptomics Analysis

This repository contains analysis performed for the Single Cell Bioinformatics course at Saarland University.

## Tools Used
- R
- Seurat
- CellChat
- tidyverse

## Dataset
10X Genomics Visium spatial transcriptomics dataset of mouse brain tissue.

## Analysis Workflow

1. Data loading using Seurat
2. Quality control and filtering
3. Normalization using SCTransform
4. PCA dimensionality reduction
5. UMAP visualization
6. Spatial clustering
7. Identification of spatially variable genes
8. Cell type annotation using label transfer
9. Spatial cell-cell communication analysis
